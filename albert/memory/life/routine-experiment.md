# Rutineeksperiment — start 2026-07-20

<!-- type=experiment source=direct-user-statement date=2026-07-20 confidence=0.95 status=active -->

## Formål

Afprøve en realistisk hverdagsrytme og følge, hvilke dele Kristian faktisk får
fulgt. Tracking er observation, ikke en karakterbog.

## Foreløbig hverdagsplan

Tiderne er målblokke og kan justeres efter virkeligheden.

- 07:00 — stå op
- 07:15–16:15 — arbejde inkl. cirka én times pause (otte arbejdstimer)
- 16:15–17:45 — fitness, cirka 1–1½ time
- 17:45–18:30 — aftensmad; tidspunktet er foreløbigt
- 18:30–20:30 — lære noget nyt og/eller arbejde med overbevisninger
- 20:30–20:45 — tjek ind med kæreste; tidspunktet er foreløbigt
- 21:00 — daglig journal/check-in

Arbejde og de øvrige blokke behandles som en hverdagsrutine, indtil Kristian
siger noget andet.

## Daglig tracking

For hver blok registreres `ja`, `delvist`, `nej` eller `ukendt` sammen med en
kort note, hvis det hjælper. En daglig status kan gives i dette korte format:

```txt
Stå op: ja/delvist/nej
Arbejde: ja/delvist/nej
Fitness: ja/delvist/nej
Læring/selvterapi: ja/delvist/nej
Kæreste-check-in: ja/delvist/nej
Journal: ja/delvist/nej
Note: ...
```

## Review-regel

Efter mindst syv registrerede dage kan vi opsummere gennemførelse, friktion og
mulige mønstre. Ingen stærke mønsterkonklusioner ud fra én eller få dage; manglende registrering holdes adskilt fra manglende gennemførelse.

## Åbne justeringer

- Den præcise kalendergentagelse og weekendadfærd skal bekræftes efter den
  første uges erfaring.
- Aftensmadens og kæreste-check-in'ets længde er foreløbige antagelser.
