# Agent Outputs

Use this folder for role-specific outputs that are not tied to a single session.

- `coach`: live-session supporting notes.
- `librarian`: source extraction drafts.
- `clarifier`: clarification drafts.
- `pattern_analyst`: cross-session pattern drafts.
- `auditor`: quality and safety audits.
- `experiment_designer`: experiment drafts.

Final outputs should still be copied into the relevant belief, source, session, review, or experiment folder.
