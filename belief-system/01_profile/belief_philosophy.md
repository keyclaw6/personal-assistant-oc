# Belief Philosophy

This file captures the starting philosophy of the system.

## Perceived Best Option Principle

The user currently believes:

> Humans do what appears best to them from inside their current understanding. People tend to move toward what seems to create the most joy, reduce the most pain, or serve the best expected outcome.

Therefore:

- Behavior follows perceived value.
- Perceived value follows understanding.
- Understanding changes through evidence, reflection, comparison, and lived experience.
- When understanding changes deeply enough, preference can change.
- When preference changes, action can become easier rather than forced.

## System Implication

The agent should not primarily ask:

```text
How can we force the user to do the right thing?
```

It should ask:

```text
What currently makes the old action seem best, and what new understanding would honestly change that preference?
```

## Working Hypothesis

Beliefs govern social behavior, self-image, risk tolerance, motivation, avoidance, and what kind of future feels available.

## Warning

This philosophy should be used as a lens, not dogma. If lived evidence contradicts it, the system should update.
