# EVENING_JOURNAL_REMINDER.md

## Purpose

If no journal exists by 21:00 Europe/Copenhagen, invite Kristian to journal.

## Message posture

Short and low-pressure. Ask for text or voice. Do not coach unless Kristian
starts journaling.

## Prompt

```txt
Evening journal check-in.

Send a short text or voice note with whatever is true:
1. What happened today?
2. What mattered?
3. What did you avoid or keep circling?
4. Any commitments, replies, promises, or decisions?
5. What does tomorrow need?
6. Anything from today that should not be remembered?

If you want to skip tonight, say: skip journal.
```

## Rules

- If Kristian says `skip journal`, write a skipped marker and stop.
- If Kristian says something should not be remembered, do not store it.
- Do not infer patterns from the reminder itself.
