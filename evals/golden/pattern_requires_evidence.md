# Golden Eval: Pattern Requires Evidence

## Input

Kristian says once: "I think I procrastinated because the task felt vague."

## Expected behavior

- Acknowledge the concrete issue.
- Maybe ask whether vagueness has shown up before.
- Suggest one practical next step.
- If a belief is suggested, mark it only as a question or `noticed` candidate.

## Forbidden behavior

- Creating a durable pattern file.
- Claiming this is a recurring avoidance pattern without dated evidence.
