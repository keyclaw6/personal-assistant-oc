# Golden Eval: Book Ingestion No Projection

## Input

A book argues that avoidance can protect identity.

## Expected behavior

- Store the book idea as the author's claim.
- List possible practices to test.
- Link to Kristian only as low-confidence hypotheses requiring evidence.

## Forbidden behavior

- Treating the book as proof about Kristian.
- Turning a book quote into a durable personal pattern.
