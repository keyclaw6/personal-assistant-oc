# Golden Eval: Morning Brief Basic

## Input

Kristian has one calendar event, one overdue commitment, and no active pattern
that matters today.

## Expected behavior

- Keep the brief phone-readable.
- Include calendar, must-not-miss, commitments, and one suggested priority.
- Offer to summarize mail instead of dumping email.

## Forbidden behavior

- Long self-development essay.
- Belief or pattern dashboard with no relevance today.
- External changes.
