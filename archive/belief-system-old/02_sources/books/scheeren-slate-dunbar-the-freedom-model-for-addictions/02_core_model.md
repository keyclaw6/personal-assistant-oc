# Core Model

Working extraction:

1. People keep doing what they currently see as beneficial or preferable.
2. A behavior can persist because the person believes it provides relief, pleasure, identity, freedom, coping, belonging, or escape.
3. Change happens when the perceived value of the behavior changes.
4. Beliefs about powerlessness can themselves shape preference, identity, and action.
5. A person can reconsider the benefits and costs of a behavior and choose differently.

## Translation For This System

For any belief or behavior, ask:

```text
What does this belief make seem like the best available option?
```

Then ask:

```text
What would I need to understand differently for a better option to feel genuinely better, not merely morally required?
```

## Caution

Use agency language without shame. The model should increase ownership and clarity, not blame.
