# Metric Definitions

See `_system/rubrics/belief_progress_scoring.md` for scoring.

## Status Movement

Useful movement includes:

- candidate to active
- active to testing
- testing to integrated
- integrated to maintenance
- maintenance to archived
- active to archived when deliberately deprioritized
- candidate to rejected when the belief is not useful

## Progress Is Not Always Status Movement

Progress can also be:

- A belief gets clearer.
- A protective function is understood.
- A book produces new candidate beliefs.
- An experiment disproves an assumption.
- The user decides a belief is not worth adopting.
