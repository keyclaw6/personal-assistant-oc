# Sessions

Every session gets its own folder:

```text
04_sessions/YYYYMMDD-HHMMSS-belief-slug/
```

Each folder should contain:

- `00_manifest.json`
- `01_transcript.md`
- `02_context_loaded.md`
- `03_interpretive_analysis.md`
- `04_deterministic_clarification.json`
- `05_belief_updates.md`
- `06_next_actions.md`
- `07_audit.md`
