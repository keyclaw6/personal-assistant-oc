# Reports

Generated reports and human review notes live here.

- `memory-report.md`: stale pages, conflicts, low confidence, missing metadata.
- `*.generated.md`: generated report slices; do not hand-edit.
- `open-questions.md`: human-maintained questions that need Kristian.
- `contradictions.md`: human-maintained unresolved contested claims.

Run:

```powershell
npm run memory:report
```
