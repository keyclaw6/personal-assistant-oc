# Raw Memory

This folder is the capture layer.

- `events/YYYY-MM-DD.jsonl`: structured append-only events.
- `daily/YYYY-MM-DD.md`: readable session and heartbeat notes.
- `inbox/`: memory candidates waiting for review.
- `conflicts/`: unresolved contradictions.
- `archive/`: retired or resolved memory.
- `_compiled/`: generated indexes and digests for startup context.

Use `memory-wiki/` for durable canonical knowledge.
