# Tasks

This folder is a local mirror and audit layer for tasks that the assistant sees from Google Keep-compatible sources.

Preferred external source:

- Google Tasks for API-backed reminders and dated todos.
- Google Keep remains acceptable as Kristian's capture surface, but direct Keep automation is optional and should stay read-first unless explicitly approved.

Use this folder for:

- task sync notes
- ambiguous task candidates
- stale task reviews
- local-only tasks that should not be pushed externally yet

Do not treat this folder as the canonical external task list once Google Tasks is connected.
