# Commitments

This folder tracks promises, follow-ups, obligations, waiting items, and expected replies.

Create one file per meaningful commitment:

```text
YYYY-MM-DD-short-slug.md
```

Use `templates/commitment.md`.

Do not store secrets or whole email bodies here. Store source identifiers, short excerpts, and links when available.

Statuses:

- `candidate`
- `active`
- `waiting`
- `done`
- `dropped`
- `contested`

Morning brief reads this folder for overdue, due-today, and waiting items.
